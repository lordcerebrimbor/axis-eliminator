# Axis Eliminator

AI-powered threat monitoring system that detects and visualizes nearby digital or geopolitical threats in real time.

## Features

* Real-time threat monitoring
* AI classification of signals
* Visualization dashboard
* Modular architecture for extensions

## Tech Stack

* JavaScript
* Node.js
* AI APIs
* HTML dashboard

## Installation

```bash
git clone https://github.com/lordcerebrimbor/axis-eliminator.git
cd axis-eliminator
npm install
npm start
```
